## Rebuilding

Every page is generated by `sitegen/` in this repo — no dependency on the cleaning pipeline:

```bash
python3 -m sitegen.build          # rebuild every page of every run, in place
python3 -m sitegen.build --check  # verify the committed HTML is what the generator produces
```
